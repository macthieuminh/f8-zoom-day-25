HyperText Markup Language

là ngôn ngữ đánh dấu siêu văn bản – ngôn ngữ chính dùng để tạo ra nội dung và cấu trúc cho trang web.

    Vai trò chính của HTML:

  • Xác định cấu trúc nội dung của trang web: tiêu đề, đoạn văn, danh sách, liên kết, hình ảnh, bảng,...
  • Tạo các thành phần như: nút, form đăng ký, menu điều hướng,...
  • Kết hợp với CSS và JavaScript để làm web đẹp hơn (CSS) và có tính tương tác (JavaScript).

Cascading Style Sheets

là ngôn ngữ dùng để tạo kiểu dáng (giao diện) cho trang web được viết bằng HTML.

    Vai trò chính của CSS:

  • Giúp trang web trở nên đẹp hơn: màu sắc, bố cục, font chữ, hiệu ứng,...
  • Tách riêng phần nội dung (HTML) và phần giao diện (CSS) → dễ bảo trì, dễ thay đổi giao diện mà không ảnh hưởng đến nội dung.
  • Kết hợp với CSS và JavaScript để làm web đẹp hơn (CSS) và có tính tương tác (JavaScript).

JavaScript

là ngôn ngữ lập trình dùng để tạo tính tương tác cho trang web – giúp website không chỉ hiển thị tĩnh (như HTML + CSS) mà còn biết phản hồi hành vi của người dùng

    Vai trò chính của CSS:

  • Xử lýtương tác: nhấn nút, di chuột, nhập dữ liệu,...
  • Thay đổi nội dung mà không cần tải lại trang.
  • Xây dựng ứng dụng web động như chat, quản lý công việc, trò chơi,...
